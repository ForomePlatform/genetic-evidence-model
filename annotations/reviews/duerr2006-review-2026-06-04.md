## Most prominent finding — MA-1, the IL-23-pathway therapeutic claim

The paper's central recommendation — *"blockade of the IL-23 signaling pathway would be a rational therapeutic strategy for IBD"* (p.3) — is captured by the annotation as `MA-1` in a `meta_assertions` block. The annotator saw the claim; the problem is the representation. **By curator decision MA-1 is retained as a meta-assertion** (a therapeutic recommendation is not itself genetic evidence — no method, no `variant_ascertainment`, no natural `target_type`), but retained is not the same as correct, and four fixes are required:

1. **No source anchor.** MA-1 carries no `source_span`; it should anchor to the p.3 sentence (PROTOCOL §4.3).
2. **Wording drift.** MA-1 says "the IL23R pathway is a rational therapeutic *target*"; the paper says the IL-23 *signaling* pathway and a therapeutic *strategy*. Receptor vs. cytokine, target vs. intervention — restate as written.
3. **Invented structure.** `meta_assertions` is a non-schema top-level key. The missing slot for a cross-item synthesizing claim is a schema gap to log as a `candidate_extension`, not to absorb into new structure (PROTOCOL §3.3, §5).
4. **Undercounted candidate.** Record **CE-DU3**. MA-1's own note withholds the candidate "pending second-paper confirmation," which misreads the rule — gaps are logged locally on first sight; the two-papers rule governs only *promotion* (PROTOCOL §5.2).

Once GE-new-1 is added, MA-1's support set should include it.

## Per-item commentary

### GE-1 — marked for edit
Strong evidentially; CE-DU1 (direction of effect) correctly exercised and de-duplicated. Edits: restate the label from *association* to *protection* and drop the "GWAS discovery —" study-type prefix; keep `knowledge_domain: [POPULATION_GENETICS]` and do **not** add HUMAN_GENETICS (curator: "it is statgen" — resolving the annotator's own uncertainty, per SR-1); remove the inactive `subdomain: GWAS`. Non-blocking migration: `STATISTICAL_GENETICS → GWAS`.

### GE-2 — marked for edit
Both assertions faithfully state the paper. Edits: make the label specific (it names neither variant nor claim); correct GE-2.A2's page citation from 3 to **2** (the OR = 0.45 phrase is in the page-2 protective-allele sentence — the one blocking span fix the curator retained); replace GE-2.A1's reconstructed token `rs11209026_p_value_Jewish: 7.95e-4` with a verbatim anchor (value is correct; now non-blocking under SR-3, retained as recommended). Migration: `STATISTICAL_GENETICS → ASSOCIATION_STUDY` — targeted replication of IL23R markers, not a fresh scan. This **diverges from ROADMAP §1.b** (which expected GWAS); the roadmap row references a "GE-8" absent from this file and is keyed to a finer decomposition. Curator endorsed ASSOCIATION_STUDY.

### GE-3 — marked for edit
The densest item. By curator ruling, `knowledge_domain → [HUMAN_GENETICS]` (family-based transmission is human genetics, not popgen; SR-1) — which made GE-3 the principled outlier and **reversed the reviewer's initial recommendation**: `mode_of_inheritance` / `mendelian_segregation` / `exact_variant` are **kept** (MOI is relevant for a within-family transmission test), and the real defect is that this conditional set activates only for HUMAN_GENETICS + **GENE**, not + VARIANT. That gap is logged as **CE-DU4** (with sub-gaps: no `COMPLEX` value for `mode_of_inheritance`; no family-based value for `subdomain`, so the forced `GWAS` is simply wrong on a transmission test). Other edits: drop `CLINICAL_EVIDENCE` from method (the evidence is the FBAT test — family-based statistical genetics); split A2, which bundles four per-phenotype clauses under a single partial anchor (§4.4); remove `subdomain`. A1's notation drift is acceptable per SR-3. Light query: `exact_variant: true` is arguable given GE-4's multiple-signals finding. Migration: `STATISTICAL_GENETICS → TRANSMISSION_DISEQUILIBRIUM_TEST`.

### GE-4 — marked for edit
Structurally the cleanest statistical item. The blocking issue is evidential, not structural: A1's claim that *rs7517847 remains significantly associated after conditioning on rs11209026* is **not stated in the main text** — the per-marker conditional result lives in table S3 (supplementary, not in the provided PDF). The main text supports independence-by-LD (r² = 0.03), the strongest combined signal (P = 3.36 × 10⁻¹³), and the aggregate multiplicity claim. Restate A1 to what the main text carries, or anchor an rs7517847-specific assertion to table S3. (Curator confirmed; this supersedes the earlier granularity query.) Also rework the method-first label to claim-first. Migration: `STATISTICAL_GENETICS → FINE_MAPPING`.

### GE-5 — approved despite flags
Originally marked for edit on the stitched A1 span; **reverted to approved** once SR-3 cleared that span, which was the sole blocking basis. A positive note: the negative result is represented as an ordinary predicate asserting absence, with no spurious polarity dimension — a clean application of the withdrawn-CE-DU2 rationale. Retained non-blocking flags: fix the stitched A1 phrase to a contiguous substring; remove `subdomain`; consider dropping the near-vacuous `variant_ascertainment` on this gene-level negative; migration `STATISTICAL_GENETICS → GWAS` (or ASSOCIATION_STUDY if table S4 is read as targeted).

### GE-6 — approved despite flags
Accepted as-is at the curator's discretion; spans are the cleanest in the file and the core cited claims are faithful. Flags preserved against the verdict for the record: `GENE_FUNCTION` looks unjustified and leaves `measurement_target`/`gene_relation` unpopulated (would drop to `[MODEL_ORGANISM]`); `IN_VITRO_EXPERIMENT` is unsupported by the in-vivo assertions; `evidence_origin: cited_prior_literature` should be the §7 field `cited_evidence: true`; the label is a role description; the "IL-23 pathway" target under `GENE` is loose. The annotator's "does GE-6 belong at all" uncertainty is resolved in the **keep** direction per §7. Migration: `IN_VIVO_EXPERIMENT → IN_VIVO`.

### GE-new-1 — approved
The anti-p40 trial item (see Missing-evidence). It records a genuine tension worth keeping in view: the antibody targets the p40 subunit (IL12B) — the same gene GE-5 found genetically *unassociated* — yet pharmacological blockade works. Its scope strain (cited therapeutic-response evidence with no fitting genetic `knowledge_domain`) is routed into **CE-DU3 as a second independent instance** alongside MA-1, which under the promotion rule strengthens the case that GEM needs a position on translational evidence.
