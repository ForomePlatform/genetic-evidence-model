\paragraph{Contributions.}
This paper makes the following contributions:
\begin{itemize}
  \item A domain-analysis-driven semantic model for genetic evidence,
        comprising a small set of core classes
        (\ScientificEvidence, \EvidenceVariable, \EvidenceAssertion)
        and their genetic specializations, together with a compact
        dimensional vocabulary structured by a conditional-activation
        mechanism.
  \item A worked mapping of the model to six published genetic-evidence
        papers spanning human, population, model-organism, and polygenic-score
        genetics, demonstrating coverage across heterogeneous evidence types.
  \item A human--AI collaborative annotation workflow, using the model
        as schema, together with an evaluation of the workflow against
        manually curated ground-truth annotations.
  \item An alignment sketch between the model and existing standards
        (FHIR Evidence, ECO, SEPIO, GA4GH GKS, IAO/OBI, PROV-O).
\end{itemize}
